# Proiect Integrare API - Random User Generator

Acest proiect reprezinta solutia pentru Tema 2, avand ca scop stapanirea unui flux de lucru profesional care combina gestionarea codului prin Git cu utilizarea tehnicilor de Prompt Engineering.

## Descrierea Aplicatiei
Aplicatia permite generarea si afisarea datelor unui utilizator aleatoriu prin interogarea Random User API.

## Detalii Tehnice si Integrare API
* **Sursa datelor**: Se utilizeaza endpoint-ul public https://randomuser.me/api/.
* **Tehnologii utilizate**: HTML5, CSS3 si JavaScript modular.
* **Monitorizare**: Sistemul de logging in consola afiseaza mesaje pentru fiecare etapa a cererii.

## Managementul Versiunilor
* Toate modificarile au fost realizate pe ramura separata numita Exersare.
* Istoricul contine minimum 5 commit-uri descriptive.
* Finalizarea proiectului este marcata de un Pull Request deschis catre main.
